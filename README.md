## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   cd server && npm install
   cd ../client && npm install
   ```

2. **Run the Application**:
   From the root directory:
   ```bash
   npm run dev
   ```
   This will start both the backend server (port 5000) and the frontend dev server (port 5173).

3. **Open the browser**:
   Navigate to `http://localhost:5173`
